# Concentra Catalog — Image Swap Guide

All challenge images live in `/images/<category>/`. Each filename maps to one challenge.
To swap an image, simply replace the file at the listed path with your own — keep the same filename and the catalog will pick it up automatically.

**Recommended image specs:** ~600×600px (square crop), JPG, under 200KB each.
